# UI2 打包與使用說明

本文件說明如何在 `ui2` 目錄打包可攜版，並說明打包後的初始預設。

## 1. 打包前準備

1. 確認已安裝 Python 虛擬環境（`ui2/venv` 或專案根目錄 `.venv`）。
2. 確認已安裝前端相依套件（`ui2/frontend/node_modules`）。
3. 建議先準備本地 LLM API（例如 `http://127.0.0.1:8080`）。

## 2. 執行打包

在 `ui2` 目錄執行：

- `build_exe.ps1`

打包完成後，輸出位置：

- 目錄：`ui2/dist/Stream Translator/`
- 壓縮檔：`ui2/dist/Stream Translator.zip`

## 3. 打包後初始預設

打包時會將 `config.example.yaml` 複製為輸出資料夾內的 `config.yaml`（若尚不存在），目前初始預設為：

- 轉錄引擎：`Qwen3-ASR`
- 轉錄模型：`Qwen/Qwen3-ASR-1.7B`
- 翻譯後端：`custom:localllm`
- LLM 模型：`localllm`
- LLM API Base URL：`http://127.0.0.1:8080`

## 4. 首次使用建議

1. 解壓 `Stream Translator.zip`。
2. 開啟輸出資料夾內的 `config.yaml`，確認本機 LLM 位址與模型名稱是否正確。
3. 啟動 `Stream Translator.exe`。

## 5. 常見調整

- 若本地 LLM 不是 `127.0.0.1:8080`，請修改：
  - `translation.gpt_base_url`
  - `translation.custom_models[0].base_url`
- 若要改成其他 Qwen3-ASR 模型，請修改：
  - `transcription.qwen3_asr_model`

---

如需改成其他打包預設，直接調整 `ui2/config.example.yaml` 後重新打包即可。
