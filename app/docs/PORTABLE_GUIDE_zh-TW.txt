Stream Translator CUDA 可攜版使用說明
====================================

一、第一次使用
--------------
1. 將完整 ZIP 解壓縮到空間足夠的資料夾，建議預留至少 15 GB。
2. 不要直接在 ZIP 壓縮檔內執行程式。
3. 執行「Stream Translator.exe」。
4. Windows 若顯示安全性警告，請確認檔案來源後選擇仍要執行。

本程式已包含 Python、PyTorch CUDA、ffmpeg 與必要套件，不需要另外安裝
Python、CUDA Toolkit、cuDNN 或 pip 套件。電腦仍需安裝相容的 NVIDIA 顯示卡驅動程式。

二、ASR 模型下載位置
--------------------
預設模型位置：
  程式資料夾\models\huggingface

可在「設定 > 模型管理」查看、開啟或變更位置。若 C 槽空間不足，可改成：
  D:\StreamTranslatorModels

變更位置不會自動搬移舊模型。需要保留時，請先關閉程式，再搬移整個
huggingface 資料夾，然後於模型管理頁填入新位置。

三、下載與刪除模型
------------------
1. 開啟「設定 > 模型管理」。
2. 在 Qwen3-ASR 或 Faster-Whisper 模型旁按「預下載」。
3. 等待狀態變成「已下載」後即可開始轉錄。
4. 不再使用的模型可在「已下載模型」區塊按「刪除」。

四、日常 App 更新
----------------
App Update 包不包含 CUDA Runtime、模型與個人設定，因此體積較小。

1. 完全關閉 Stream Translator。
2. 備份 config.yaml（正常更新不會覆蓋，但仍建議保留備份）。
3. 將 App Update ZIP 解壓縮後的內容覆蓋到原程式資料夾。
4. 請勿刪除 models、config.yaml、output、srt 與 logs。
5. 重新啟動程式並確認版本與功能。

只有發行說明明確標示「Runtime 必須更新」時，才需要重新下載完整版或 Runtime 包。

五、常見問題
------------
程式無法啟動：
- 確認 ZIP 已完整解壓縮。
- 確認防毒軟體沒有隔離 EXE、DLL 或 _runtime 內檔案。
- 更新 NVIDIA 顯示卡驅動程式。

模型一直重新下載：
- 到模型管理頁確認目前模型路徑。
- 確認該資料夾可寫入，且沒有被清理工具刪除。

更新後設定消失：
- 確認覆蓋更新時沒有用更新包內檔案取代 config.yaml。
- 可從備份還原 config.yaml。

模型或程式佔用空間過大：
- 在模型管理頁刪除不用的 ASR 模型。
- 模型資料夾可移到其他磁碟。
- llama 模型不包含在本程式包內，請自行管理其 GGUF 檔案。
